Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base operator of the multiply operation    [número] [requerido]
  -l, --limit    limit operator of the multiply operation   [número] [requerido]

  Execute: node app -b=5 -l=10

  Result:
  5 x 1 = 5
  ...
  5 x 10 = 50
